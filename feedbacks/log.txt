Geral,Explicação,que projeto é esse?,Este projeto é sobre criar um sistema que pode comparar textos de maneira eficiente e precisa. Ele tem como objetivo principal ajudar a detectar plágio  identificar paráfrases e comparar diferentes tipos de conteúdo.  ### Como funciona?  O sistema usa duas abordagens diferentes para comparar textos:  1. **Léxica**:    - **O que é?**: Analisa os padrões de palavras nos textos  usando técnicas como TF-IDF (Term Frequency-Inverse Document Frequency) e n-gramas.    - **Por que?**: Essa abordagem é rápida e dá uma boa ideia de como as palavras se repetem nos textos.  2. **Semântica**:    - **O que é?**: Analisa o significado dos textos usando embeddings (representações numéricas dos textos) com a biblioteca `sentence-transformers`.    - **Por que?**: Essa abordagem é mais complexa  mas consegue capturar similaridades mesmo quando as palavras usadas são diferentes.  ### Como o sistema é organizado?  O projeto está dividido em várias partes:  - **app/**: Aqui temos a interface do usuário  que permite que você digite ou carregue textos e veja os resultados.
